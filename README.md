# CMS SERVER

Implemented by [Nest](https://github.com/nestjs/nest).

## Description

The server for [CMS](https://github.com/chtocode/CMS).

## Installation

```bash
npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

# debug mode
$ npm run start:debug
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
